<br>


Crystall-X Scanner merupakan proyek aplikasi mobile yang dibuat untuk memenuhi kebutuhan klien perusahaan CV Fiesto Informatika. Fitur pada aplikasi ini adalah melakukan scan pada barcode dan mengirimkan product id, longtitude, serta latitude ke url yang telah ditentukan. 


## Persyaratan

* Flutter
* IDE/Text editor
* Android emulator

## Mengunduh dan Menjalankan Proyek

1. **Duplikat Repositori**
    ```bash
    git clone <repository-url>
    ```

2. **Masuk ke Direktori Proyek**
    ```bash
    cd crystall-x-scanner
    ```

3. **Unduh Semua Dependensi**
    ```bash
    flutter pub get
    ```

4. **Jalankan Program**
    ```bash
    flutter run
    ```

### Membangun proyek

Untuk membangun proyek, silahkan baca dokumentasi resmi Flutter untuk membangun aplikasi Flutter. Berikut merupakan pranala untuk membangun aplikasi Android: "https://docs.flutter.dev/deployment/android".
